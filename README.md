My neovim config.

## Installation

### Prerequisites

- Neovim (+0.8.0)

### Install

```bash
pip install pynvim
git clone https://github.com/hmgle/nvim.git ~/.config/nvim
git clone --depth 1 https://github.com/wbthomason/packer.nvim\
  ~/.local/share/nvim/site/pack/packer/start/packer.nvim
nvim +PackerSync
```
